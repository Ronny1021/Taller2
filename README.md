# Taller 2 - Práctica de HTML y CSS

Este repositorio contiene el desarrollo del segundo taller de tecnologías web, enfocado en el diseño de un catálogo de películas y el manejo de formularios.

##  Características
* **Diseño Personalizado**: Uso de una paleta de colores en azul oscuro y azul claro.
* **Optimización de Imágenes**: Implementación de `object-fit: cover` para mantener la proporción de los pósters.
* **Estructura Semántica**: Uso correcto de etiquetas HTML5.

##  Tecnologías Utilizadas
* **HTML** y **CSS**.
* **Git**: Gestión de ramas y Pull Requests.
* **Markdown**: Documentación del proyecto.
